# Codex Context

`.codex-context` stores lightweight project context for long-running Codex work.

It is not a Codex configuration folder. It does not change Codex behavior by itself.

It is only a local record for the current project: task direction, progress, changed files, validation, checkpoints, and handoff notes.

If you no longer want these project notes, delete `.codex-context/`.

If you also want to uninstall the Skill, delete the Skill directory itself.
