# Runtime Binaries

M18.2 expects the Windows packaging step to place these files here before `npm run package:cutter-desktop:windows`:

- `cutter-api-sidecar-x86_64-pc-windows-msvc.exe`
- `mixlab-searchd-x86_64-pc-windows-msvc.exe`
- `ffmpeg.exe`
- `ffprobe.exe`

The large binary files are intentionally not committed in this scaffold.
