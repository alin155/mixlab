# M14.1 剪辑端主工作台重构设计

## 目标

M14.1 将剪辑端从“功能页面集合”重构为“文案驱动的视频选段工作台”。

剪辑师的核心任务不是浏览模块，而是高效完成：

```text
搜文案 -> 定位视频 -> 拖选文案 -> 剪切这段 -> 沉淀为本地可复剪素材
```

产品原则：

- 视频是主视觉。
- 文案是主操作系统。
- 本地素材优先。
- 拖选文案生成视频选区。
- 选中后直接剪切，不再暴露“片段篮”或“待剪清单”概念。
- 剪切完成后进入本地素材库，并作为可复剪素材参与后续搜索。

## 一级导航

剪辑端保留 5 个一级导航：

1. 素材定位
2. 剪切任务
3. 本地素材
4. 公共素材库
5. 设置

### 素材定位

默认首页。承载“搜、选、剪”的高频主路径。

职责：

- 搜索本地素材与公共原素材。
- 优先展示本地素材结果，再展示公共原素材结果。
- 打开候选视频进行画面验证。
- 展示完整自然文案。
- 支持点击文案定位视频。
- 支持拖选文案生成视频选区。
- 支持“剪切这段”直接创建剪切任务。

### 剪切任务

后台剪切任务中心。

职责：

- 查看等待中、剪切中、已完成、失败任务。
- 查看进度、错误原因、输出文件。
- 重试失败任务。
- 打开已完成视频或所在文件夹。

它不再承担“提交前清单”的职责。

### 本地素材

剪辑师个人的本地可复剪素材库。

职责：

- 展示已经剪切完成的本地素材。
- 展示可参与搜索和再次剪切的本地视频。
- 未来支持本地视频导入和本地预处理。
- 支持整理、删除、备注、诊断本地索引。

本地素材不是“整段复用成品库”，而是更近、更快、更小的二级原素材库。

### 公共素材库

保留一级导航，当前页面布局先不重构。

职责：

- 浏览管理端发布的公共 ready 原素材。
- 让剪辑师了解公共素材库的课程、讲师、标签、封面、说明。
- 从公共素材进入单个视频详情子页面。
- 未来支持将明确要用的公共视频下载或同步为本地素材，提高预览、seek 和剪切速度。

公共素材库不是默认首页，也不是日常搜选主路径。

### 设置

低频配置页。

职责：

- 公共素材库连接状态。
- 本地工作区路径。
- FFmpeg 状态。
- 默认剪切模式。
- 本地并发数。
- 诊断入口。

## 子页面与页面关系

### 原视频详情

原视频详情不再作为一级导航。

它是公共素材库和本地素材点击单个视频后的子页面，用于查看单个素材详情、视频、完整文案和基础信息。

素材定位工作台中的选段区可以复用详情组件，但产品语境不同：

- 从素材定位进入时，用户目标是快速选段并剪切。
- 从素材库进入时，用户目标是浏览和了解素材详情。

### 不再暴露提交前清单

不再作为用户可见概念。

系统内部可以继续使用 clip-list 或任务模型，但剪辑师不需要先加入清单再提交。拖选完成后，用户看到的主动作是：

```text
剪切这段
```

## 素材定位工作台

### 顶部搜索区

顶部是全局核心搜索框。

搜索范围：

- 全部
- 本地素材
- 公共原素材

视频方向筛选：

- 全部
- 横版
- 竖版

状态摘要：

- 本地可用素材数量。
- 公共可用素材数量。
- 当前剪切中数量。

### 搜索结果区

搜索结果按来源分组，顺序固定为：

1. 本地素材
2. 公共原素材

每个结果展示：

- 封面。
- 标题。
- 来源标签：本地素材或公共原素材。
- 视频方向标签：横版、竖版、方形或未知。
- 命中数量。
- 命中文案摘要。
- 时长。

点击结果后，不跳到独立详情页，而是在素材定位工作台中打开该视频。

### 视频主视觉区

视频用于判断画面是否可用，是主视觉。

交互：

- 点击文案位置后，视频跳到对应时间点。
- 默认从命中文案所在时间点前 0.5 到 1 秒开始预览。
- 拖选文案后，视频跳到选区起点。
- 支持播放当前选区。

### 文案操作区

文案以完整自然文本呈现。

必须满足：

- 不显示可见时间码。
- 不显示可见 segment 边界。
- 不显示字幕块列表。
- 不做表格化文案。
- 搜索命中词高亮。
- 当前选区高亮。

系统内部仍然可以用不可见的 segment span 保存映射关系：

```text
字符选区 -> 起止 segment -> begin_ms / end_ms
```

剪辑师只看到自然文案，只像选择普通文字一样操作。

### 文案交互

统一操作模式：

```text
点击文字 = 定位视频
拖选文字 = 选择视频片段
松开鼠标 = 生成当前选区
剪切这段 = 创建剪切任务
```

不提供并列的“设为起点”“设为终点”“第一次点击起点、第二次点击终点”等模式。

拖选规则：

- 选区起点落在哪个 segment，就取该 segment begin_ms。
- 选区终点落在哪个 segment，就取该 segment end_ms。
- 多个连续 segment 只生成一个视频片段。
- 空选区不允许创建剪切任务。

### 选区操作条

拖选完成后显示轻量操作条：

```text
已选中一段文案 · 预计 xx 秒
播放预览   剪切这段   取消
```

不显示默认时间码。必要时可以在展开详情中显示起止时间。

点击“剪切这段”后：

- 立即创建剪切任务。
- 当前页面不跳转。
- 任务进入后台队列。
- 剪辑师可以继续搜索、继续选段。
- 剪切完成后，本地素材库自动出现该素材。

### 任务状态轻提示

素材定位页只展示轻量任务状态，不抢主工作流。

示例：

```text
剪切中 1 · 等待 3 · 完成 8
查看全部任务
```

完整管理进入“剪切任务”页面。

## 视频方向规则

所有素材入口必须识别视频方向。

判断规则：

```text
width > height  -> 横版
width < height  -> 竖版
width = height  -> 方形
缺少宽高       -> 未知
```

布局规则：

- 横版视频：视频吸附在上方，文案操作区在下方。
- 竖版视频：视频吸附在左侧，文案操作区在右侧。
- 方形或未知：默认使用横版布局，并显示对应方向标签。

公共素材库、本地素材、素材定位搜索结果都必须展示方向标签，并支持横版/竖版筛选。

## 本地素材参与搜索

素材定位默认同时搜索：

- 本地可复剪素材。
- 公共原素材。

本地结果优先显示。

M14.1 本地素材搜索的最低要求：

- 已完成剪切并带有文案映射的本地素材可以参与搜索。
- 如果本地素材只有一段完整 selected_text，则它可以作为单段可复剪素材进入统一选段流程。
- 没有文案或时间映射的本地文件暂不进入素材定位搜索，保留在本地素材页做后续预处理。

公共原素材继续只展示和搜索 ready 视频。

## 公共素材下载与同步

M14.1 只保留产品入口和结构约束，不要求完整实现下载同步。

未来同步规则：

```text
公共原素材 -> 同步到本地素材 -> 本地素材参与搜索和剪切
```

同步价值：

- 提升视频预览速度。
- 提升 seek 定位速度。
- 提升剪切读取速度。
- 降低公共盘或 NAS 压力。
- 支持弱网或离线工作。

## 非目标

M14.1 不做：

- 本地视频导入完整流程。
- 本地视频 ASR 预处理流水线。
- 公共素材下载同步的完整实现。
- 复杂语义搜索或 AI 推荐。
- 项目级素材管理。
- 高级批量剪切清单排序。
- 公共素材库页面布局重构。

## 验收标准

1. 剪辑端默认首页为“素材定位”。
2. 一级导航为“素材定位、剪切任务、本地素材、公共素材库、设置”。
3. 原视频详情不再作为一级导航，只作为素材库子页面或工作台内嵌选段区。
4. 素材定位搜索结果按“本地素材、公共原素材”分组展示。
5. 搜索结果可按“全部、本地素材、公共原素材”和“全部、横版、竖版”筛选。
6. 搜索结果展示视频方向标签。
7. 点击搜索结果后，在素材定位工作台中打开视频和完整文案，不跳离主工作台。
8. 横版视频使用视频上、文案下布局。
9. 竖版视频使用视频左、文案右布局。
10. 文案呈现为完整自然文本，无可见时间码、无可见字幕块、无可见 segment 边界。
11. 点击文案位置可以定位视频播放位置。
12. 拖选文案可以生成一个连续视频选区。
13. 选区操作条主按钮为“剪切这段”。
14. 点击“剪切这段”直接创建后台剪切任务，不进入待剪清单。
15. 创建剪切任务后仍停留在素材定位页，允许继续搜索和选段。
16. 剪切任务页展示等待中、剪切中、已完成、失败和重试。
17. 剪切完成的素材进入本地素材库。
18. 公共素材库页面保留当前图库布局，并保留作为一级导航。
19. 公共素材库和本地素材点击单个视频可进入原视频详情子页面。
20. 系统用户界面不再出现“片段篮”或“待剪清单”作为主流程概念。
